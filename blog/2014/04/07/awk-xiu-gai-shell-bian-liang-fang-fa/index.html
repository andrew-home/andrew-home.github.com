Awk 修改 Shell 变量方法

Apr 7th, 2014

  1. 使用文件重定向：

     #!/bin/bash 

 var=1
 awk 'BEGIN {print 2>"tmp"}'
 var=$( cat tmp )

  2. 使用eval

     #!/bin/bash

 eval $(awk 'BEGIN {print "var=2"}')
 echo $var
